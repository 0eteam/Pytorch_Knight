## 文件结构

```
--Environment       强化学习外部环境：用于获取输入、状态、奖励，产生截图和语音
--Module            网络模型：用于学习网络
--Agent             智能体：用于输出动作
--Tool              额外工具
--prototype         通用模型示例
--doc               文档
--example           运行示例
--dataset_autosave  模型自动保存位置
--dataset_manual    人工指导数据集
  
--main.py           程序入口
--conf.yaml         配置文件
--validate.py       验证环境是否完整有效
```
